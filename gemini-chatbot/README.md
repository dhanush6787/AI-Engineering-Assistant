# AI Engineering Assistant Chatbot

A Python-based conversational web application built for engineering students, powered by the Google Gemini API and Streamlit.

## Features
- Real-time conversational memory.
- Minimalist, responsive UI.
- Secure environment configuration.

## How to Run Locally
1. Install dependencies: `pip install -r requirements.txt`
2. Set your environment variable: 
   - Windows: `$env:GEMINI_API_KEY="your_actual_key"`
   - Mac/Linux: `export GEMINI_API_KEY="your_actual_key"`
3. Run the app: `streamlit run app.py`
